# Eldralis: Primal Awakening – RPG System for SillyTavern ( Beta 1 SFW )

A complete Dark Fluff anthro RPG system for SillyTavern, powered by DeepSeek V4 Flash Thinking.

---

## Files

- `Eldralis_System_Prompt_SFW.txt` – The Narrator System Prompt (SFW)
- `Eldralis_System_SFW.json` – Core World Info (anatomy, mechanics, NPCs, bestiary, quests, etc.)
- `Eldralis_Knowledge_SFW.json` – World & Environment WIs (regions, societies, narrative tools, etc.)

---

## Features

- Dark Fluff (cozy + horror) narrative style
- D&D 5e based mechanics with visible dice rolls
- Dynamic Focus System – the model adapts to the scene type
- 24 anthro species with full D&D 5e stats
- Spiderweb Quest Matrix – non-linear, branching quests
- Full character creation (16 steps) with D&D 5e stats
- English naming convention (even in Hungarian output)
- Hard Knowledge Firewall – NPCs know only what they should
- All NSFW content removed – Discord safe

---

## Installation

1. Import the System Prompt into your SillyTavern character.
2. Load both JSON files as World Info (Lorebook).
3. Start the adventure!
